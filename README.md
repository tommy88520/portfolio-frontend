## **Using 📦**

1. Install Packages

```
yarn install
```

2. Start Project

```
yarn dev
```

3. If you using git, delete the existing folder .git after cloning (open `git bash` or other terminal)

```
rm -rf .git
```

## **Options ✍️**

1. Check lint

```
yarn lint
```

2. Fix lint

```
yarn lint:fix
```

3. Check prettier

```
yarn prettier
```

4. Fix prettier

```
yarn prettier:fix
```

5. Fix lint and prettier

```
yarn format
```
